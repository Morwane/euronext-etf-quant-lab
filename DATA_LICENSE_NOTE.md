# Data Note

This repository does not ship raw LSEG / Refinitiv market data.

The code is designed to be reproducible by users with an active LSEG Workspace session. Raw pulls are saved locally under `data/raw_lseg*`, and processed CSV outputs are saved under `data/processed/`; both are ignored by Git.

The included figures and reports are illustrative project outputs. For public redistribution, verify that your market-data license permits sharing derived analytics.
